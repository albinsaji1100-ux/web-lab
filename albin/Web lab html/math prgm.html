JavaScript String and Math Functions

String Methods

Let's explore some common String functions in JavaScript:

  • length: Returns the number of characters in a string.
  • toUpperCase() / toLowerCase(): Changes the case of the string.
  • substring(start, end): Extracts part of a string.
  • indexOf(): Finds the index of a substring.
  • replace(): Replaces part of the string with another.
  • trim(): Removes whitespace from both ends.
